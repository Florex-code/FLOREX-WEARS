AVENOIR (React + Vite)

Run:
  npm install
  npm run dev

Open:
  http://localhost:5173

Features:
- Multi-page store (Home, Shop, Product, About, Contact)
- Cart Drawer + persisted cart
- Search Modal (Ctrl/Cmd + K)
- Authentication (Login/Register) stored in LocalStorage
- Protected routes (Checkout, Account)
- Admin dashboard (CRUD products) - admin only
- Dark/Light theme toggle
- Toast notifications
